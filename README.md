# Windhawk Blur Dock

A Windhawk Blur Dock for Windows 11 — centered, floating, blurred dock-style taskbar with hidden system tray.
<img width="1919" height="1040" alt="image" src="https://github.com/user-attachments/assets/842ac45d-1e66-4150-8282-3e7b0a487274" />

## Features
- Centered taskbar with fixed side margins
- Rounded corners (16px) + translucent blur background
- Hidden system tray and background stroke
- Auto-width gripper

## Install
1. Install [Windhawk](https://windhawk.eu/)
2. Install the taskbar styling mod this targets (specify mod name/ID here)
3. Paste `config.yaml` contents into the mod's style config
